# Bundled LuaUnit

Version 3.4 available from either [GitHub][LuaUnit34GitHub] or
[LuaRocks][LuaUnit34LuaRocks]. It's under 30k when gzipped. Bundled to make
`temper test -b lua` faster and more lightweight.

[LuaUnit34GitHub]: https://github.com/bluebird75/luaunit/blob/LUAUNIT_V3_4/luaunit.lua
[LuaUnit34LuaRocks]: https://luarocks.org/modules/bluebird75/luaunit/3.4-1
